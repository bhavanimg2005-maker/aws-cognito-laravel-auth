# AWS Cognito Authentication using Laravel

A secure authentication system built using **AWS Cognito** and the **Laravel** framework. The project demonstrates cloud-based identity management using OAuth 2.0 authentication and AWS managed authentication services.

---

##  Project Overview

This application integrates AWS Cognito with Laravel to provide secure user authentication without implementing a custom authentication system. Users can register, sign in, sign out, verify their email, and recover passwords using AWS Cognito Hosted UI.
